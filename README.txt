How to run:

Open up the chaser project file. Run the program using the debug solution configuration(default)

How to use:

The program shows the implementation of shaodws using shadow volume and shadow mapping.
There are a couple of scene that are in the rotation list. Using PAGE DOWN will move up
the list of scenes. PAGE UP will move down the scenes.
1. The visible space using shadow mapping
2. Above view of scene with shadow using shadow mapping
3. Output of Z-Buffer from lights point of view
4. The visible space using shadow volume



Camera Controls:
The camera controls work for scenes 1, 2 and 3

w - Move camera foward
s - Move camera backwards
a - Turn counter clockwise
d - Turn clockwise


Ball Controls :
The ball controls work only for scene 4

Arrow keys rotates the ball
