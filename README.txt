How to use:

The program shows the implementation of shaodws using shadow volume and shadow mapping. 
The first scene is shadows using the shadow mapping technique. The second scene is 
shadows using the shadow volume technique. Scenes can be change by using PAGE UP to change 
from first to second and PAGE_DOWN to change from second to first.


Camera Controls:

w - Move camera foward
s - Move camera backwards
a - Turn counter clockwise
d - Turn clockwise


Ball Controls :

Arrow keys rotate ball
